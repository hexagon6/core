fontSize

Returns a function that takes an object containing fontSize and theme properties. This function is meant to be used with styled-components. Call this function within your styled-component's template literal like so:
Source:
  • lib/fontSize.js, line 3
Examples
const MyComponent = styled.div`
  ${fontSize()}
`
const MyComponentWithFallback = styled.div`
  ${fontSize({ breakpoint1: '1rem', breakpoint2: 18})}
`
